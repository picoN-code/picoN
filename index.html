picoN

Programming in Plain English.

picoN is a lightweight programming language designed to make coding simple, readable, and approachable. Instead of learning complex symbols, brackets, and semicolons, write programs using natural English-like statements.

Download picoN Learn More

Why picoN?

Readable by Humans

Most programming languages look like this:

JavaScript
let age = 25;
if (age >= 18) {
    console.log("Adult");
}
picoN
store age as 25.

if age is at least 18 then
  say Adult.
end if.

The code reads almost like plain English.

Minimal Symbols

picoN intentionally keeps punctuation to a minimum. Only two symbols are used:

Period (.) Comma (,)
No braces No parentheses No semicolons No complicated syntax

Factorial Function

factorial.pico
define factorial using n
  if n is 0 then
    give back 1.
  otherwise
    store temp as n.
    subtract 1 from temp.
    store result as run factorial with temp.
    multiply result by n.
    give back result.
  end if.
end define.

store answer as run factorial with 5.
say answer.

Install picoN

Getting started takes only a few minutes.

Requirements: Python 3.x

  1. Download picoN
  2. Extract the files
  3. Open the project in VS Code
  4. Create a .pico file
  5. Write your program
  6. Run: python picon.py myprogram.pico

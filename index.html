picoN

A programming language that lets you code in plain English.

picoN is a lightweight, beginner-friendly programming language designed to make programming easier to learn and easier to read. Instead of complicated symbols, braces, and syntax rules, picoN uses clear English-like statements that focus on logic and readability.

Features

💬

English-Like Syntax

Code reads like natural English statements.

✂

Minimal Punctuation

Only commas and periods. No braces or semicolons.

📦

Variables & Types

Numbers, text, and booleans with simple syntax.

🔢

Arithmetic

Add, subtract, multiply, and divide with words.

🎚

User I/O

Ask for input and display output naturally.

🧪

Conditions

If/otherwise logic with clear comparisons.

🔁

Loops

Repeat, while, and count loops for iteration.

⚙️

Functions

Define functions with parameters and return values.

See picoN in Action

Simple, clean, and easy to understand.

Hello World
note My first picoN program.

store name as text World text.

say text Hello text, name.
Conditions
store age as 20.

if age is at least 18 then
  say Adult.
otherwise
  say Minor.
end if.
Loops
count i from 1 to 5
  say i.
end count.
Functions
define greet using name
  say text Hello text, name.
end define.

run greet with text Alex text.
Arithmetic
store score as 10.

add 5 to score.

if score is greater than 10 then
  say High score.
end if.
Data Types
store age as 25.
store name as text Alex text.
store active as yes.

Why picoN?

Many programming languages are powerful, but can be difficult for beginners because of their syntax.

"Programming should be about learning logic, not memorizing symbols."

With picoN, code reads almost like plain English, making it easier to understand and teach.

Installing the Extension

  1. Click the download button.
  2. Open Visual Studio Code.
  3. Press Ctrl+Shift+P.
  4. Run Extensions: Install from VSIX...
  5. Select picon-vscode.vsix.
